# 🌟 My First Git Project — What I Learned Today 🚀

Hello everyone! 👋  
This is my **first Git project** that I uploaded on GitHub.  
It might look small, but for me it was a **big achievement** and an amazing learning experience.  

---

## 📝 What I Did
✔️ Created a new project folder in my system  
✔️ Initialized Git using **`git init`**  
✔️ Added my files using **`git add .`**  
✔️ Made my first commit with  
   > `git commit -m "Initial commit - branch-demo project"`  
✔️ Connected it to GitHub using **`git remote add origin`**  
✔️ Finally pushed it using **`git push -u origin main`**  

---

## 💡 Key Learnings
1️⃣ **Git Basics** → initializing, adding, committing, and pushing changes  
2️⃣ **Branching** → importance of creating separate branches and merging them  
3️⃣ **Conflict Resolution** → fixing code conflicts while merging  
4️⃣ **GitHub Upload** → how to move a local project safely to GitHub
